== Installation ==
<div id="tab-installation" class="plugin-installation section">
	<h2 id="installation-header">Installation</h2>
	<h4>Automatic installation</h4>
<p>Automatic installation is the easiest option as WordPress handles the file transfers itself and you do not need to leave your web browser. to<br>
do an automatic install of, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.</p>
<p>In the search field type “Once Cart Items Remover" and click Search Plugins. Once you’ve found out about this plugin, click here. Most importantly, you can install it by simply clicking “Install Now”.</p>
<h4>Manual installation</h4>
<p>The manual installation method involves downloading our plugin and uploading it to your web server via your favorite FTP application.
